# Logo Setup Instructions

## Add the FooW Clan Logo

1. Save your logo image as: `public/foow-logo.png`
2. The logo will automatically appear throughout the website

The logo is already integrated in:
- Navbar (top left)
- Footer
- Favicon (browser tab icon)

## Alternative: Use the image URL directly

If you want to use the image from a URL, the code is ready to accept either:
- Local path: `/foow-logo.png`
- External URL: `https://your-image-url.com/logo.png`

Just place the image in the `public/` folder and it will work automatically.
